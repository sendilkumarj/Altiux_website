IoT Solution Accelerators

We understand the importance of agility and speed in today’s fast changing world. Altiux provides a solution accelerator tool kit that enables customers to develop innovate products and solutions. Our toolkit comprises of device Connectivity software, Edgeware platform, Analytics framework, Mobility framework and Reference hardware implementations.

_________

  • Connectivity middleware
    BoXPwr

    As the paradigm of smart connected devices expands into areas such as manufacturing facilities, health care, hotels, homes, energy grids and transportation systems; a vast number of hitherto unconnected legacy devices as well as new breed of intelligent devices are expected to hook up to the internet. Enabling connectivity on resource constrained devices ranging from heart rate monitors to temperature & humidity sensors is a huge challenge.

    Altiux’s connectivity middleware BoxPwr connects constrained and legacy devices to the IoT network infrastructure. It is a bidirectional connectivity middleware with configurable variants powering a wide range of applications

    BoXPwr is available in four variants; Standard, Micro, Nano and Pico. They can be deployed in applications ranging from low power devices like wearables and health monitors to complex applications requiring cloud connectivity like smart home gateways, intelligent buildings and manufacturing automation.

    These stacks are currently ported on leading chip-sets like Texas Instruments TI MSP430, ARM Cortex M0/3/4 and Intel® QuarkTM and supports OS platforms like Contiki, TinyOS,FreeRTOS, Linux and WindRiver.

  • Reference IoT
    hardware

    Altiux provides reference hardware designs and blue prints that helps customers fast track their product development, cut development costs and minimise the integration risks. Our reference gateway enables seamless connectivity between devices/sensors and other systems on the Internet.

    The key features of our platform include:

    - The ability to connect to non IP/legacy devices using wired/wireless protocols.
    - Multiple flavours of the embedded software stack, with varying functionality and code footprint to cater to multiple gateway configurations, including dedicated devices or mobiles.
    - Multi service cloud gateway, supports single sign on functionality and delivers IoT services, mobility services, entertainment services and information services. It can be deployed as a SDK, as a virtual appliance or as a physical appliance.
    - A cloud based software framework that eases management i) gateways connected to the platform ii) services/ contacts/ devices/ contents connected to the gateways iii) services/ contacts/ devices/ contents connected to the end users of the gateways.
    - We provide rich workspaces for service providers to provision/manage the devices, applications and services.
    - Dedicated mobile cloud stack comprising of Personal Portal workspace, Mobile Gateway Agent for integrated management, Mobility service cloud gateway and mobile embedded software.
    - An application enablement framework comprising of various standalone connectors for managing data streams within the platform as well as integrating it with external applications.
    - Support for multiple deployment configurations including public cloud, private clouds, hybrid clouds and community clouds.

  • Edgeware platform
    IoTEE

    As the number of connected devices in the IoT network grows, there is significant complexity involved in deploying new devices and managing existing ones. Data from field devices has to be quickly routed to the correct application. Altiux IoTEE is a Platform to manage IoT devices and data with support for multiple device connectivity protocols.

    The IoTEE platform provides a number of benefits:

    - Simplified development
    - Common infrastructure
    - Ease of manageability
    - Dedicated support

    Our IoTEE platform is engineered to be flexible and fit different deployment scenarios from device and data management functionalities for small deployments to more complex functions for larger deployment.

  • Mobility
    Framework

    Altiux’s Mobility solution is a Cross-Platform (Android and iOS) framework to develop mobile IoT applications for IOS, Android, BB and Windows devices. It includes Notification Framework (Web Socket, GCM, MQTT, APN), IOT Platform Connector, IOT Service enablement component, BIG Data Analytic Service enablement, Mobile Analytics.
